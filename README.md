# Xenteck Business Bottleneck Audit

Static hero page for the Business Bottleneck Audit with a Make.com webhook form.

## Quick setup

1. Open `index.html`.
2. Replace `YOUR_MAKE_WEBHOOK_URL` in the form `data-webhook` attribute.
3. Update the business phone number in the hero CTA and footer.

## Deploy

- Vercel can deploy this as a static site with no build step.
- Connect the GitHub repo to Vercel and it will serve `index.html`.
