# Xenteck Speed Audit

Static hero page with a fast audit form that posts to a Make.com webhook.

## Quick setup

1. Open `index.html`.
2. Replace `YOUR_MAKE_WEBHOOK_URL` in the form `data-webhook` attribute.
3. Update the business phone number in the hero CTA.
4. Update the fallback email in the footer.

## Deploy

- Vercel can deploy this as a static site with no build step.
- Connect the GitHub repo to Vercel and it will serve `index.html`.
